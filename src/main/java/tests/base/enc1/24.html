СВУмк-2
Более современный вариант снайперской винтовки СВДм 2, в котором для уменьшения габаритов использована компоновка «булл-пап». Обладает более высоким по сравнению с «родительницей» боевым темпом стрельбы при меньшей массе, причём точность боя на малой и средней дальностях осталась примерно той же. Сегодня пользуется вполне заслуженной популярность в войсках и среди сталкеров.
