Чёрный ястреб
«Чёрный ястреб» — израильский крупнокалиберный пистолет, обладающий чрезвычайно сильной отдачей, что, впрочем, с лихвой компенсируется его убойной силой. Пистолет предназначен в основном для охоты и спортивной стрельбы. Редкая вещь в Зоне.

Настоящая «карманная пушка» — большая, тяжёлая и обладающая чрезвычайно высокой убойной силой. Разрабатывался как специализированное оружие для охоты на среднюю и крупную дичь, а также для спортивной стрельбы по силуэтным мишеням на дальности от пятидесяти метров. Из-за цены, веса и габаритов, и при том что в каждый из перечисленных параметров можно вместить два других пистолета, особого распространения в Зоне не получил. Не предназначен для скрытого ношения.
