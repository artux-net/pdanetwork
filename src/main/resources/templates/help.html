Использование обработчика команд

Ниже представлены команды, позволяющие выполянть различные действия с персонажем в контексте прохождения историй.

С помощью команд, можно задать параметры, значения, репутацию, заметки, деньги, предметы, статусы прохождения сюжета и выполнить сброс.

Синтаксис

{ "actions":{ "команда":["набор1","набор2"] } }

Наборы могут разделяться двоеточием ':'


Команда add

В зависимотсти от структуры набора, выполняет добавление предмета, значения или параметра.

"2:2"

Выполнит добавление предмета с id 2 в количестве 2 штук, то же, что и add_items

"value:2"

Выполнит добавление значения 'value' = 2, то же, что и add_value

"param"

Добавление параметра 'param', то же, что и add_param


Команда add_param

Выполняет добавление параметра.

"param"

Добавление параметра 'param'


Команда add_value

Выполняет добавление значения. Если значение уже присутствует, то к нему применится операция сложения.

"value:2"

Добавит к значению 'value' 2


Команда add_items

Выполняет добавление предмета. Предметы доступны в энциклопедии, id будет указан в конце url страницы предмета. Если предмет уже присутствует, то к нему применится операция сложения. Оружие и броня не складывается.

"3:2"

Создаст новый предмет, так как оружие (id=3 относится к пистолетам) не складывается.


Команда remove

Удаляет предметы, значения и параметры.

"3:2"

Удалит предмет в количестве 2 штук. Необходимо заметить, что будут удалены все предметы с таким id.

"valueOrParam"

Удалит значение и параметр 'valueOrParam'


Команда =

Задает значение

"3:2"

Задаст значение c именем '3'


Команда +

Добавляет репутацию в группе или значение.

Набор для группы выглядит как 'relation_1:5'

Где 1 - номер группы, 5 - номинальное значение репутации. Дальше 5 попадает под формулу Репутация += номинал за квест Х (0.5-(0.5 Х rep/max_rep)) +0.5 Х номинал за квест

Виды групп (название : номер)

Иначе команда работает как add_value.


Команда -

Команда берет отрицательное значение от заданного положительного, в остальном аналогична '+'

"relation_1:2"

Вычтет номинальные 2 из репутации.


Команда *

Умножение значения

"value:2"

Умножит 'value' на 2


Команда money

Добавляет и вычитает денежные средства

"-1000"

Вычтет 1000 RU

"1400"

Добавит 1400 RU


Команда xp

Добавляет и вычитает опыт

"-1000"

Вычтет 1000 XP

"1400"

Добавит 1400 XP


Команда note

Создает заметку

"title:content"

Создаст соответствующую заметку

"content"

Создаст заметку с названием 'Новая заметка' и содержанием 'content'


Команда achieve

**В разработке**


Команда reset

Если наборы отсутствуют, сброс информации о прохождении, установка денежных средств в 0 RU

В наборах:

"2"

Сбросит историю 2 в 1 главу, 0 стадию

"relation_1"

Установит отношение к группировке 1 в 0.


Команда reset_current

Сбросит текующую историю в 1 главу, 0 стадию. В наборах не нуждается.


Команда set

Устанавливает персонажа в истории

"story:0:1:2"

Задаст состояние проходения 0 истории на 1 главу, 2 стадию
